Dasharo OSF vPub Spring 2022

26 May 2022

4 PM UTC - last haker standing

Calendar Invites: iCal / Google


  • Dasharo OSF vPub Matrix room in Dasharo Workspace (Matrix) with embedded Jitsi conference using Element.
    Following configuration similar to virtual FOSDEM, we added integration of Jitsi to our Dasharo OSF vPub Matrix channel.
    If you don't have Matrix account, here is a quick guide on how to get started with Element & Matrix or feel free to use Jitsi directly (link below).
  • Jitsi Conference Link - please click to join Jitsi audio (optional video) conference via web browser or mobile app
  • Dasharo Workspace (Matrix) - join #dasharo-osf-vpub:matrix.org optional Matrix channel for text chat
  • Dasharo OSF vPub Newsletter - if you would like to be informed about future vPub events feel free to sign up to our newsletter


5th edition of vPub is approaching! We will be discussing lots of interesting topics with many great guests!
